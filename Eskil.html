Eskil: A Tk frontend to Diff

The first major application I wrote in Tcl/Tk was this tool. Writing it is what made me learn and enjoy Tcl. (I already liked Tk but found Tcl annoying)
It is similar to "TkDiff".

Features

Highlights changes within a line.
CVS/RCS diff. CVS conflict merge. Some ClearCase support.
View patch file. Print to postscript.
Directory diff. "Clip diff".
